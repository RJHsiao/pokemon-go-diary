# pokemon-go-iv-api
Restfulapi server for getting your pokemon's IV in JSON (Pokemon Go)

## Install
`npm install`

## Usage
`curl -H "Content-Type: application/json" -X POST -d '{"username":"xxxxx", "password": "xxxxx"}'  http://localhost:8887/pokemon/list`
